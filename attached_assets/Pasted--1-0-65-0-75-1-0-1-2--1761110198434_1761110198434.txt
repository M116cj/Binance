優化建議
1. 提高信号質量
•  調整方向：
	•  增加 τ (如 0.65 → 0.75)：提高信心度，過濾低概率信号。
	•  增加 κ (如 1.0 → 1.2)：確保收益顯著高於成本。
	•  增加 θ_up 和 θ_dn (如 0.6% → 0.8%, 0.4% → 0.5%)：提高標記門檻，確保信號更可靠。
•  適用場景：風險厌惡型交易者，追求高勝率。
•  影響：信号數量減少，但質量提升，適合穩定市場。
2. 增加信号數量
•  調整方向：
	•  降低 τ (如 0.75 → 0.65 或 0.55)：允許更多低概率信号。
	•  降低 κ (如 1.2 → 1.0 或 0.8)：放寬收益要求。
	•  降低 θ_up 和 θ_dn (如 0.6% → 0.3%, 0.4% → 0.2%)：降低標記門檻。
•  適用場景：風險偏好型交易者，追求高交易頻率。
•  影響：信号數量增加，勝率可能下降，適合波動較大市場。
3. 適應市場波動
•  高波動市場：
	•  增加 θ_up 和 θ_dn (如 0.6% → 1.0%, 0.4% → 0.6%)：適應更大價格波動。
	•  保持或略降低 τ (如 0.65 → 0.60)：確保信号數量。
	•  保持 κ 不變或略降低 (如 1.0 → 0.9)：放寬收益要求。
•  低波動市場：
	•  降低 θ_up 和 θ_dn (如 0.6% → 0.3%, 0.4% → 0.2%)：適應小幅價格變動。
	•  保持或略提高 τ (如 0.65 → 0.70)：確保信号質量。
	•  保持 κ 不變或略提高 (如 1.0 → 1.1)：確保收益覆蓋成本。
4. 時間窗口優化
•  建議：
	•  根據交易風格選擇合適的 horizon_minutes：
		•  超短線（5分鐘）：適合高頻交易，需降低 θ_up 和 θ_dn。
		•  短線（10分鐘）：平衡頻率與質量，推薦默認設置。
		•  中短線（30分鐘）：適合穩健交易，建議提高 θ_up 和 θ_dn。
	•  測試不同時間窗口組合（如 [5, 10] 或 [10, 30]），根據回測結果選擇最佳組合。
5. 策略選擇與參數組合
•  保守型策略（高質量、低頻率）：
	•  設置：θ_up = 0.006, θ_dn = 0.004, τ = 0.75, κ = 1.20
	•  優化：監控勝率和收益成本比，適當增加 θ_up 和 θ_dn 至 0.8% 和 0.5%。
•  平衡型策略（質量與頻率平衡）：
	•  設置：θ_up = 0.006, θ_dn = 0.004, τ = 0.65, κ = 1.00
	•  優化：根據市場波動調整 θ_up 和 θ_dn，保持 τ 和 κ 穩定。
•  激進型策略（高頻率、接受較低勝率）：
	•  設置：θ_up = 0.006, θ_dn = 0.004, τ = 0.55, κ = 0.80
	•  優化：降低 θ_up 和 θ_dn 至 0.3% 和 0.2%，監控最大回撤。
6. 訓練與回測優化
•  訓練參數：
	•  保持 focal_gamma = 1.5，若模型過擬合，嘗試降低至 1.0。
	•  使用 calibration_method = "isotonic"，監控 calibration_ece 是否接近 0.04，若偏差大，考慮切換至 “sigmoid” 校準。
•  回測參數：
	•  延長 days_back 至 60 天，獲取更穩健的統計數據。
	•  啟用 enable_embargo = True，避免過擬合。
	•  調整 cooldown_minutes 至 15-20 分鐘，平衡交易頻率與重疊風險。
7. 監控與動態調整
•  關鍵指標：
	•  信号頻率：目標每天 5-20 個信号，根據策略調整 τ 和 κ。
	•  勝率：保守型目標 70%+，平衡型 60%+，激進型 50%+。
	•  收益成本比：確保平均值 > 1.0，保守型目標 1.5+。
	•  夏普比率：目標 > 1.0，優化風險調整後收益。
	•  最大回撤：控制在 5%-10% 以內，激進型可放寬至 15%。
•  動態調整：
	•  實時監控市場波動，通過 UI 側邊欄動態調整 θ_up 和 θ_dn。
	•  每周回測一次，根據 pr_auc (目標 > 0.75) 和 hit_at_top_k (目標 > 0.70) 微調參數。
8. 程式碼與實時調整
•  參數配置：
	•  在 main.py 中設置 UI 默認值，確保與策略一致。
	•  在 backend/inference_service.py 中檢查層級閾值（A級/B級），確保 τ 和 κ 邏輯正確。
	•  在 backend/models/labeling.py 中驗證標記參數，確保 θ_up 和 θ_dn 適配市場。
•  UI 優化：
	•  提供快速切換策略的按鈕（保守/平衡/激進）。
	•  顯示實時監控指標（信号頻率、勝率等），方便用戶即時調整。
9. 其他建議
•  市場適應：定期分析市場波動性（如 7 天平均波動率），動態調整 θ_up 和 θ_dn。
•  風險管理：設置止損線（如 -0.5%），結合 κ 確保交易風險可控。
•  數據質量：確保輸入數據（價格、成交量等）無異常值，影響標記和預測。
